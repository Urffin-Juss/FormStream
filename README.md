# FormStream

**FormStream** — веб-система для автоматизации сбора, проверки и обработки клиентских анкет с последующей подготовкой данных для выгрузки в учетную систему предприятия.

Проект разрабатывается в рамках дипломной работы по направлению веб-разработки.

## Назначение проекта

Система предназначена для замены ручного процесса обработки анкет, которые ранее заполнялись через Google Forms и затем вручную проверялись, дополнялись и переносились в учетную систему.

FormStream должен обеспечить:

- ввод анкет через веб-интерфейс;
- локальное хранение данных;
- автоматическую проверку обязательных полей;
- проверку адреса и GPS-координат;
- ручную обработку спорных анкет;
- формирование печатных форм;
- экспорт данных в формат, совместимый с 1С.

## Основная идея

Текущий бизнес-процесс содержит несколько ручных этапов:

1. Торговый представитель заполняет анкету.
2. Сотрудник вручную проверяет данные.
3. Данные переносятся в печатную форму.
4. Оператор вручную вводит данные в 1С.

FormStream автоматизирует этот процесс и сокращает количество ручных операций.

## Планируемые роли пользователей

### Администратор

- управление пользователями;
- управление справочниками;
- просмотр всех анкет;
- настройка системы.

### Оператор

- просмотр анкет;
- проверка и редактирование данных;
- подтверждение или отклонение анкет;
- формирование документов;
- экспорт данных.

### Торговый представитель

- заполнение анкеты;
- отправка данных на обработку.

## Основные модули системы

- `accounts` — пользователи, роли и доступы;
- `forms_app` — анкеты, клиенты и карточки заявок;
- `validation` — проверка данных, адресов и GPS;
- `exports` — формирование печатных форм и файлов выгрузки;
- `core` — общая логика проекта.
